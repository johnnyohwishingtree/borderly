# Thailand Pass Manual Guide Assets

This directory contains visual assets and screenshots for the Thailand Pass manual submission guide.

## Required Screenshots

1. **01-registration.png** - Thailand Pass homepage with registration button
2. **02-account-creation.png** - Account creation form
3. **03-personal-info.png** - Personal information form section
4. **04-travel-info.png** - Travel details form section
5. **05-accommodation.png** - Accommodation information form
6. **06-health-info.png** - Health and vaccination section
7. **07-document-upload.png** - Document upload interface
8. **08-submission-confirmation.png** - Final submission and QR code generation

## File Naming Convention

- Use descriptive names with leading numbers for ordering
- PNG format preferred for screenshots
- Maximum resolution: 1920x1080
- Compress images to keep file sizes reasonable

## Usage

These screenshots are referenced in the `submissionGuide` section of the THA.json schema and displayed in the React Native app's manual submission walkthrough feature.
