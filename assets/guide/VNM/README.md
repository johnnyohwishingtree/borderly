# Vietnam e-Visa Manual Guide Assets

This directory contains visual assets and screenshots for the Vietnam e-Visa manual submission guide.

## Required Screenshots

1. **01-portal-entry.png** - Vietnam e-Visa homepage
2. **02-nationality-selection.png** - Nationality dropdown selection
3. **03-personal-info.png** - Personal information form
4. **04-passport-details.png** - Passport information section
5. **05-travel-info.png** - Travel purpose and entry details
6. **06-accommodation.png** - Accommodation information form
7. **07-contact-info.png** - Contact and emergency information
8. **08-document-upload.png** - Passport and photo upload interface
9. **09-payment.png** - Payment processing page
10. **10-evisa-download.png** - e-Visa download confirmation

## File Naming Convention

- Use descriptive names with leading numbers for ordering
- PNG format preferred for screenshots
- Maximum resolution: 1920x1080
- Compress images to keep file sizes reasonable

## Usage

These screenshots are referenced in the `submissionGuide` section of the VNM.json schema and displayed in the React Native app's manual submission walkthrough feature.
