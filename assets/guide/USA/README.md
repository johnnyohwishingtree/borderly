# USA ESTA Manual Guide Assets

This directory contains visual assets and screenshots for the ESTA manual submission guide.

## Required Screenshots

1. **01-disclaimer.png** - ESTA disclaimer and terms acceptance
2. **02-application-type.png** - Individual vs. group application selection
3. **03-personal-info.png** - Personal information form
4. **04-passport-info.png** - Passport details section
5. **05-birth-info.png** - Birth information form
6. **06-address-info.png** - Home address and contact details
7. **07-parent-info.png** - Parent information section
8. **08-employment-info.png** - Employment details form
9. **09-travel-info.png** - Travel plans and U.S. address
10. **10-contact-info.png** - Emergency contact in U.S.
11. **11-eligibility-questions.png** - Detailed eligibility questionnaire
12. **12-review.png** - Application review page
13. **13-payment.png** - Payment processing ($21 fee)
14. **14-esta-confirmation.png** - ESTA authorization confirmation

## File Naming Convention

- Use descriptive names with leading numbers for ordering
- PNG format preferred for screenshots
- Maximum resolution: 1920x1080
- Compress images to keep file sizes reasonable

## Usage

These screenshots are referenced in the `submissionGuide` section of the USA.json schema and displayed in the React Native app's manual submission walkthrough feature.
