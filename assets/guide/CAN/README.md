# Canada eTA Manual Guide Assets

This directory contains visual assets and screenshots for the Canada eTA manual submission guide.

## Required Screenshots

1. **01-eligibility-check.png** - Canada eTA eligibility checker
2. **02-start-application.png** - Application start page
3. **03-personal-info.png** - Personal information form
4. **04-passport-info.png** - Passport details section
5. **05-birth-info.png** - Birth information form
6. **06-address-info.png** - Home address and contact details
7. **07-employment-info.png** - Employment information section
8. **08-travel-info.png** - Travel plans and purpose
9. **09-eligibility-questions.png** - Health and background questions
10. **10-review.png** - Application review and confirmation
11. **11-payment.png** - Payment processing (CAD $7 fee)
12. **12-eta-confirmation.png** - eTA approval confirmation

## File Naming Convention

- Use descriptive names with leading numbers for ordering
- PNG format preferred for screenshots
- Maximum resolution: 1920x1080
- Compress images to keep file sizes reasonable

## Usage

These screenshots are referenced in the `submissionGuide` section of the CAN.json schema and displayed in the React Native app's manual submission walkthrough feature.
