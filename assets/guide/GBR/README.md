# UK ETA Manual Guide Assets

This directory contains visual assets and screenshots for the UK Electronic Travel Authorisation manual submission guide.

## Required Screenshots

1. **01-eligibility-check.png** - UK ETA eligibility checker
2. **02-account-creation.png** - GOV.UK account creation
3. **03-login.png** - Sign in to existing account
4. **04-passport-info.png** - Passport and personal details form
5. **05-travel-info.png** - Travel dates and purpose section
6. **06-contact-info.png** - Contact information form
7. **07-security-questions.png** - Security questions setup
8. **08-background-check.png** - Background and eligibility questions
9. **09-payment.png** - Payment processing (£10 fee)
10. **10-eta-confirmation.png** - ETA approval and download

## File Naming Convention

- Use descriptive names with leading numbers for ordering
- PNG format preferred for screenshots
- Maximum resolution: 1920x1080
- Compress images to keep file sizes reasonable

## Usage

These screenshots are referenced in the `submissionGuide` section of the GBR.json schema and displayed in the React Native app's manual submission walkthrough feature.
